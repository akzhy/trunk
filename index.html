Trunk.css

Trunk.css is a lightweight framework that provides a basic grid system based on Materialize.

DOWNLOAD | Github

What is it for ?

Trunk is useful for projects that doesn't require larger frameworks and only requires a base file that includes a grid and the most basic stylings. Trunk is not for UI designing, it only provides the grid and a few custom styles.

Light Weight

~4kb

Not for UI designing

Easily Customizable

Includes .less files

Usage

Download the files from Github and include it as follows

<link href="dist/trunk.min.css" rel="stylesheet" type="text/css"/>

Or include from jsdelivr

<link href="dist/trunk.min.css" rel="stylesheet" type="text/css"/>

That's it now you can use all the functionalities of Trunk

Grid

Grids are created in the same way as that of Materialize.css

A basic grid can be created as follows

<div class="container">
    <div class="row">
        <div class="col s12 m6 l4 xl3"></div>
    </div>
</div>

Trunk follows the 12 grid format, so the above code results in a div that is full width on smaller screens, half on medium devices, one third on large screens and one fourth on extra large devices

Break points
Device Screen Width
Smaller Screens (s) <= 600px
Medium Screens (m) <= 992px
Large Screens (l) <= 1200px
Extra Large Screens (xl) > 1200px

Equal height columns can be created by adding the class flex to .row

Code

<div class="container">
    <div class="row flex">
        <div class="col s12 m4"></div>
        <div class="col s12 m4"></div>
        <div class="col s12 m4"></div>
    </div>
</div>

Buttons

Buttons can easily be created with the btn class. There are four different types of buttons, adding the btn class alone will produce the default style, other styles are primary, secondary and danger

<button class="btn">Default Button</button>
<button class="btn primary">Primary Button</button>
<button class="btn secondary">Secondary Button</button>
<button class="btn danger">Danger Button</button>

Forms

All inputs are normalized and are given basic styles

<form>
    <div class="row">
        <div class="col s12 m5">
            <label class="full-width">
                <span class="label-text">Name</span>
                <input type="text" name="name" class="full-width"/>
            </label>
        </div>
        <div class="col s12 m5">
            <label class="full-width">
                <span class="label-text">Email</span>
                <input type="email" name="email" class="full-width"/>
            </label>
        </div>
        <div class="col s12 m5">
            <label class="full-width">
                <span class="label-text">Select Option</span>
                <select class="full-width">
                    <option selected>Choose Option</option>
                    <option>Option 1</option>
                    <option>Option 2</option>
                </select>
            </label>
        </div>
        <div class="col s12 m5">
            <label class="full-width">
                <span class="label-text">Subject</span>
                <input type="text" name="subject" class="full-width"/>
            </label>
        </div>
        <div class="col s12 m10">
            <label class="full-width">
                <span class="label-text">Message</span>
                <textarea name="message" class="full-width"></textarea>
            </label>
        </div>
        <div class="col s12 m10">
            <button class="btn primary">Submit</button>
        </div>
    </div>
</form>

Utility Classes

There are a few utility classes, such as

  • .full-width - For applying 100% width
  • .hidden - For applying display none
  • .center - On .row.flex, this will center the columns horizontally
